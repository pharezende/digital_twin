The Digital Twin App is a containerized RAG application that lets users interact with a digital representation of a person’s profile. It consumes profile documents, stores their embeddings in ChromaDB, retrieves relevant context by exploiting an embedded model, and generates answers using LangChain and LLM (running via OLLAMA) through a Gradio web interface. The project also supports persistent storage and Prometheus monitoring.
